### Etapes pour tester:

Installer =>
    *[node.js](https://nodejs.org/en/)*

Cloner le projet avec tapant la commande:
```
    git clone https://github.com/zlorgoncho1/Markdown-App.git
 ``` 
      
Installer les dépendances avec:
```
    npm install
```
      
Lancer le server:
 ```
    npm start
```
Puis accéder à 
 ```
    localhost:3000
```
